Info

This tool force-aligns your audio with a transcript and returns word-level timings using WhisperX on the server.

Paste text or upload a .txt file and pick an alignment language that matches it.

After aligning, you can preview playback, click words to jump in the audio, and download JSON, SRT, or CSV.

The server needs ffmpeg on PATH to decode audio. GPU is optional but speeds up alignment when available.
